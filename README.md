
## Hotkeys
#### Left Arrow: jump 5 sec. back
#### Right Arrow: jump 5 sec. forward
#### Up Arrow: volume increase
#### Down Arrow: volume decrease
#### Space: play/pause
#### K: play/pause
#### Esc: leave fullscreen mode
